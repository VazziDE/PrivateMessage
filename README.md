# PrivateMessage
PrivateMessage is a PocketMine-MP Plugin to send a private message to other player.

*How to get the plugin?*

1. Download the plugin from here.
2. Drag the plugin into your plugin order on the server.
3. Start the Server.
4. Have fun with the plugin.

The plugin is easy to use:

Send a private message to player:

  /message [PlayerName] [Message] or use
  /msg [PlayerName] [Message]
  
Reply to a message:

  /reply [Message] or use
  /r [Message]
  
The Reply function allows you to reply to a private message you got from an other player.
You can also send again a message to the player, you send a message before.
